# Staff RJA - Sistema de Gestión

Este proyecto es una herramienta administrativa para la gestión de locutores, moderadores y directores de Radio Juvenil Adventista (RJA).

## Características

- **Autenticación**: Integrada con Supabase Auth.
- **Gestión de Personal**: Tablas dinámicas para Moderadores, Locutores y Directiva.
- **Generación de Certificados**: Herramienta interactiva en Canvas para crear reconocimientos.
- **Reportes**: Sistema de registro mensual de actividades.

## Instalación

1. Clona este repositorio.
2. Abre `index.html` utilizando un servidor local como **Live Server** para evitar problemas de CORS con las políticas de seguridad de Supabase.
